JavaScript Basics

Javascript is a cross-platform,o object-oriented scripting language. JavaScript is extremely popular for a vareity of reasons. It is small and lightweight language allowing maximumflexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment(a web browser or Node server), it can be connected to the objects these environments to provide programmatic control over them.

  • Variable Declaration:JavaScriptvariables are containers for storing data valuies - imagine a cup you fill with coffee, the cup holds the cofee a variable holds a value. All JavaScript variables must be identifies unique names. These unique names are called identifiers. var x;
  • Variable Assignment:Assignment operators assign values to JavaScript variables = our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x = 10
  • Data Types:Data types are an important concept; to be able to operate on variables you need to know the data type.   • Loops:There are many different kinds of loops in every language, but they all enssentially do the same thing: they will repeat an action a number of times. Imagine you have to run a mile, while you run around the track four times and then you stop. thats a loop!
